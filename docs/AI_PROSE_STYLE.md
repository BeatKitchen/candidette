# AI Prose Style Guide

Rules for all prose output — website copy, blog posts, client-facing documents, social media drafts. The goal: writing that sounds like a person wrote it, not a language model.

## Banned Phrases (hard rules — never use these)

- "Here's the thing" / "But here's the thing"
- "Let me break it down" / "Let's dive in" / "Let's delve into"
- "And honestly?" / "Honestly,"
- "Let that sink in" / "Read that again" / "Let that land"
- "This changes everything"
- "...and that's rare"
- "...and that matters" (as a rhetorical kicker)
- "The takeaway" (as a section header or transition)
- "What you need to understand is"
- "Let me explain" / "Let me be clear"
- "But the scary part?"
- "You just said the quiet part out loud"
- "You're not imagining things"
- "Stay in that for a moment"
- "Raise your hand if"
- "Mind you,"
- "Here's the/where/why/another" as a sentence opener — lead with the actual point instead

## Banned Structures (hard rules)

- **"It's not X. It's Y."** as rhetorical flourish. OK when genuinely distinguishing two things ("It's not a PAC, it's a 527"). Not OK as false-profundity ("It's not a campaign. It's a movement.").
- **"Not A. Not B. But C."** — the triple-beat reveal. Never.
- **Staccato fragment pileup for dramatic effect**: "A phrase. A fragment. A word." This is the single most recognizable AI cadence pattern.

## Cadence Rules (softer — use judgment)

- **Avoid the setup-explain-kicker paragraph shape** where every paragraph ends with a tidy punchline sentence. Real writing doesn't land every paragraph like a TED talk.
- **Vary paragraph endings.** Some should trail off. Some should stop mid-thought and let the next paragraph pick it up. Not every paragraph needs a bow on it.
- **Don't over-parallel.** If you write "X does A. Y does B. Z does C." — break the symmetry. Make one longer, drop one, reorder.
- **Bullet lists aren't always the answer.** If a list has 3 items and each is a full sentence, it's probably prose pretending to be a list. Write it as a paragraph.
- **Don't throat-clear.** "Worth noting," "important to understand," "the key insight here" — just say the thing. If it's important, the reader will notice.
- **Em dashes are fine** but watch density. More than two per paragraph starts reading as a tic.

## Why This Exists

AI-generated text has recognizable tells — punchy parallel structures, predictable paragraph rhythm, rhetorical false-contrast, and signature filler phrases. Even when the content is accurate and useful, these patterns undermine credibility. A political consultancy selling strategic communication cannot afford to sound like a chatbot.

Before finishing any prose output, re-read for rhythm. If every paragraph has the same shape, break some. If you catch yourself reaching for a rhetorical device from the banned list, rewrite the sentence to just state the point directly. Plain, slightly irregular prose reads as human.
