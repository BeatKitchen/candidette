# Site Audit - candidette.com

## Overview
The current Squarespace homepage contains the following sections:

1. **Hero**: Title "CANDIDETTE" with tagline "Run for office as yourself." and subheading "Unapologetically progressive. Proudly authentic."
2. **Intro narrative**: Several paragraphs explaining the "Candidettes" concept (outsiders running for office) with images interspersed.
3. **About Liz**: A personal introduction under "Competitive Grassroots Campaign Management for Candidettes." describing her background, values, and quirky details (tarot, shoes, perfumes).
4. **Services**: List of core services (Strategic Planning & Messaging, Field Operations, Fundraising, Social Media Management, Event Planning, Budget Management) each with an accompanying image.
5. **Contact**: Basic contact info (email, cell, location) at bottom.
6. **Site links**: navigation includes "What's a Candidette?" and "Let's Work Together" and an email link.

## Content Worth Migrating
- The core concept of "Run for office as yourself" and the definition of Candidette.
- Personal storytelling elements about Liz (outsider, corporate background, force multiplier language).
- Service list can be kept but reformatted for clarity.
- Contact details.

## Content to Rewrite
- Block paragraphs are long and can be streamlined for web readability.
- Quirky list (shoes, tarot, etc.) could be moved to an "About" page or persona section, trimmed for professionalism or used sparingly for whimsy.
- Some jargon-free rewriting for SEO (more specific keywords for municipal/state, etc.).

## Missing/Must-Haves for New Site
- Stronger above-the-fold CTA (e.g., "Schedule a call" button) and summary of value props.
- Testimonials or case studies placeholder.
- Clear menu structure (Home, About, Services, Contact, Blog maybe).
- Mobile-friendly design and accessible fonts/colors.

## SEO Baseline
- Existing meta descriptions appear generic, needs custom copy.
- Headings not properly nested (repeated "CANDIDETTE" H1 etc).
- Image alt text likely missing (squarespace typically generates non-descriptive alt text).

## Design Observations
- Current site uses a generic serif font and muted palette; it feels clean but lacks personality and confidence.
- Images appear to be stock graphics; real photos would add authenticity.
- The layout is standard Squarespace with wide text blocks and no accent colors.

## Recommendations
1. **Structure**: Build separate pages (Home, About, Services, Contact, possibly Blog). Use concise, scannable sections with headings and bullets.
2. **Tone**: Preserve whimsy and humor in the About page and in small design touches; ensure the homepage communicates competence and trustiness within first screen.
3. **Design**: Choose a theme with modern typography, generous white space, and vibrant accent color (coral/teal). Add playful icons or hand-drawn accents for whimsy.
4. **Action**: Add clear CTAs like "Get a pricing estimate" or "Let's talk" visible in header and at end of each page.
5. **SEO**: Update metadata, use keywords like "state legislature campaign management," "municipal campaign consultant for women," etc. Optimize heading structure.
6. **Imagery**: Source candid photos or creative illustrations that align with the whimsical yet confident tone.
